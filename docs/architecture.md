<h1 align="center">Architecture</h1>

### Standard directory structure

```
├── publish/ 
└── src / 
    ├── assets/                     // static resource directory
    ├── common/                     // general class library directory
    ├── components/                 // public component directory
    ├── router/                     // routing configuration Directory
    ├── store/                      // State management directory
    ├── style/                      // General CSS directory
    ├── utils/                      // Tool function directory
    ├── views/                      // Page component directory
    ├── App .vue
    ├── main .ts
    ├── shims-vue .d .ts
├── __tests__/                         // Unit test directory
├── index.html
├── tsconfig.json                   // TypeScript configuration file
├── vite.config.ts                  // Vite configuration file
└── package.json

```
